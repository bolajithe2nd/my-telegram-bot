> # SET UP AND INSTALLATION

```bash
npm init -y
npm install -D typescript ts-node nodemon
npm i express body-parser cookie-parser compression cors lodash dotenv bcryptjs jsonwebtoken
npm i -D @types/express @types/body-parser @types/cookie-parser @types/compression @types/cors @types/lodash @types/bcryptjs @types/jsonwebtoken
npm install prisma --save-dev
npx prisma
npx prisma init
```
